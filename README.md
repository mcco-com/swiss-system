<h1>Features</h1>
</div>

- Creating tournaments using the Swiss system 
- Add an unlimited number of players to the rounds 
- Adding the number of tables and automatically assigning players to them 
- Automatic selection of the player's piece color after each round 
- The next round will be created automatically after the previous one ends. 
- Copy tournament information as a JSON file 
- Chess puzzles for you 
- It works without needing the internet. 
- Chess tips via notifications 
- Home screen widget with chess tips 
- Dynamic theme
- Material 3 (Not all of it) 
- etc.
